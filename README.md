# Autodiscovery-V2.0
Autodiscovery-V2.0 is a Tool which helps you discover applications and processes running on your linux machine.

# Requirements

  1.  Python 2.6 or Above
  2.  Node.js
  3.  mysql (Yet to be implemented)
